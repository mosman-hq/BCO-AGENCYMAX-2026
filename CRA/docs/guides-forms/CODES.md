# 1 Introduction

## 1.1 About this document

This document contains various codes used by the Canada Revenue Agency's Charities IT system and is to be used in conjunction with the data dictionary.

## 1.2 Summary of Changes

1) Category/sub-category codes: These codes were heavily refactored.
2) Country: Added "QZ" – Other unknown countries
3) Program area codes no longer stored on the T3010 after the 2019 filing period This is due to the BIS (basic information sheet) being discontinued as of May 2019 implementation.
